Project Overview – Ohm Desai’s Indie Game Hub

Project Name

Ohm Desai's Indie Game Hub

Project Overview

  • Function: This website will serve as a central hub for Ohm Desai’s independent game development projects, helping promote current and future titles.
  • Purpose: To provide details about each game, offer downloadable or playable demos, share updates, and build a following.
  • Intended Users: Gamers, indie game fans, publishers, friends, and future collaborators.
  • Website Content: Game descriptions, development blog posts, downloadable builds, press kits, and a contact form.

    • Planned Pages & Designs

    1. Home

    • Purpose: Landing page with game highlights and latest updates
    • Audience: General visitors
    • Content: Hero section, featured games, recent blog posts
    • Interactivity: Clickable banners, nav buttons

    2. Games

    • Purpose: Showcase of Ohm’s game projects
    • Audience: Gamers and publishers
    • Content: Game title, description, screenshots, play/download links
    • Interactivity: Gallery sliders, download buttons

    3. Dev Log

    • Purpose: Share blog-style updates about development progress
    • Audience: Followers and other devs
    • Content: Posts, images, maybe videos
    • Interactivity: Expand/collapse post previews

    4. About

    • Purpose: Tell Ohm’s story, inspirations, and background
    • Audience: Visitors, fans
    • Content: Bio, photo, creative philosophy
    • Interactivity: Optional: animated timeline or hover effects

    5. Contact

    • Purpose: Let visitors reach out via form
    • Audience: Press, potential collaborators, fans
    • Content: Input fields (name, email, message)
    • Interactivity: Form with JavaScript validation

    Dynamic Functionality

    • Image Gallery: Planned for the “Games” page. Interactive image carousels for each game. Example: itch.io
    • Expandable Blog Posts: Dev Log page will use JavaScript to collapse/expand entries
    • Contact Form: On the Contact page with validation
    • Dropdown Nav + Hover Effects: Clean nav bar with subtle animations
    • jQuery UI: Used for potential accordion-style dev log

    Note: All data is static (no user accounts or saved data). Interactions will be client-side only due to project scope.
