Bugfixing

I was right, the Spotify widget was indeed buggy. I had to do some digging around and I found out that while its true that lyric websites can use the Spotify API to play samples of songs. Spotify canceled their API service that allowed for that. Thats why the 30 second previews weren't working. I've fixed it now but its taken a lot of effort. Long story short, I needed to change the previews to use itunes, then setup a backend server to host the API calls to itunes. Then I had to adjust a ton of the code to make sure it worked with the new API and the code isn't just hard coded to only work on specific websites, but can work on any website that uses the API. There were so many bugs to fix today, I've totally lost track. I guess thats why people do devlogs, to make up where memory fails.

More css and assets

So I added some fun css to convert the rain effect to a firefly effect. A little off topic but I love the firefly bushes in minecraft, they have such nice ambient sounds and they're pretty to look at. I like rain a lot, but I wanna be a little more original, anyways the vibe is still relatively cozy I think. I also added a video background and website music. For the record I didn't make either, both assets were from free use websites, and the respective owners are credited in the footer of the website. I actually had to track down the creator of the video myself, they weren't listed on pixabay's website as the orignal creator. Maybe one day I'll make my own video background and music, but for now I think the current assets go a really long way in making the website feel more alive. I wanted to do a cafe theme originally for my website, but it sorta just turned into this instead. It's very discord/steam/spotify inspired. Oh and the video background actually has a css filter applied to it. I learned that trick in web dev class. I think it works well with the existing color scheme.

Whats next?

The website is now in a state where I can call it ready to be launched. I'll probably continue to update it and add more stuff as I learn more. But for now I need to focus on getting a better job, and updating my resume before that. I wonder if I should add the code of the spotify widget I made to the projects page. I think it would be a good idea, but ahh I'm so tired right now, I'll think about it later. Surely this is my last all nighter, surely! (haha) This was a lot of fun to do, I'll consider this website in a state of "soft launch". My friend wanted me to help with his game he's making. The genre of the game isn't super intriguing to me, but despite the genre, any game could have a good story. Might be a fun project to work on.
